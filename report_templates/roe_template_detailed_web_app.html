4. Methodology & Focus Areas

The Tester will follow the OWASP Web Security Testing Guide (WSTG). Testing will focus on, but is not limited to, the OWASP Top 10 vulnerabilities, including Injection (SQL, Command), Broken Authentication, Sensitive Data Exposure, XXE, Broken Access Control, Security Misconfiguration, Cross-Site Scripting (XSS), and Insecure Deserialization.

Testing of business logic flaws is (in/out of) scope.
